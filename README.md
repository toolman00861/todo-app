# 待办事项 + 番茄钟应用

这是一个使用WPF实现的小型待办事项和番茄钟应用，帮助用户管理任务并使用番茄工作法提高工作效率。

## 功能特点

- **待办事项管理**
  - 添加、删除和标记完成待办事项
  - 显示待办事项的优先级
  - 显示待办事项的创建时间
  - 跟踪每个待办事项的番茄钟完成情况

- **番茄钟计时器**
  - 工作时间（默认25分钟）
  - 短休息时间（默认5分钟）
  - 长休息时间（默认15分钟）
  - 每完成4个番茄钟后自动进入长休息
  - 支持暂停、继续和重置功能

- **待办事项和番茄钟整合**
  - 选择待办事项后可以直接开始番茄钟
  - 番茄钟完成后自动更新待办事项的番茄钟计数

## 技术实现

- **开发框架**: WPF (.NET Core)
- **设计模式**: MVVM (Model-View-ViewModel)
- **项目结构**:
  - `Models`: 数据模型
  - `ViewModels`: 视图模型
  - `Converters`: 值转换器
  - `MainWindow.xaml`: 主界面

## 如何使用

1. **添加待办事项**
   - 在左侧输入框中输入待办事项标题
   - 点击"添加"按钮或按回车键添加

2. **管理待办事项**
   - 点击复选框标记待办事项为完成/未完成
   - 选择待办事项可以将其与番茄钟关联

3. **使用番茄钟**
   - 点击"开始工作"按钮开始一个番茄钟周期
   - 工作时间结束后会自动进入休息时间
   - 可以使用"暂停"、"继续"和"重置"按钮控制番茄钟

## 开发环境要求

- .NET 6.0 或更高版本
- Visual Studio 2022 或其他支持WPF的IDE

## 如何构建和运行

1. 克隆仓库到本地
2. 使用Visual Studio或命令行打开解决方案
3. 构建并运行项目

```bash
dotnet build
dotnet run
```
